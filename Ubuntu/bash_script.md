# Bash Script

Bash Script is executable file that can run various command. Bash Script file extension is `.sh`

## Create Bash Script

`nano sample.sh`

inside sample.sh insert this line

```sh

#!/bin/bash

echo "hellow world"

```

save and exit nano editor

## Compile Bash Script

`chmod +x ./sample.sh`

## Run Bash Script

`./sample.sh`
