# Shortcut

## Open file

`cmd` + `shift` + `o`

## Go to line

`cmd` + `l`

## Go to Declaration

`cmd` + `b`

`cmd` + `button1 click`

## Page Up

`fn` + `press up`

## Page Down

`fn` + `press down`

## Go next tab

`cmd` + `shift` + `[`

## Go previous tab

`cmd` + `shift` + `]`

## Fold code block

`cmd` + `-` or

`cmd` + `+`
